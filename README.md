## 움직이는 자동차

### 요구 사항

- [x] 자동차는 이름을 가지고 있다.
- [x] 자동차는 0~9사이의 랜덤 값을 통해 4 이상인 경우 전진, 3 이하일 경우 멈춘다.
    - 랜덤 값을 뽑을 때마다 4 이상인 경우 위치 +1, 아닌 경우 그대로이다.

### 코드 요구 사항

- [x] indent depth 2 이하
- [x] 3항 연산자 사용 X
- [x] else, switch/case 사용 X
- [x] 함수 길이 15라인 이하

## 게임 실행

### 요구 사항

- [x] 주어진 횟수 동안 n대의 자동차는 전진 또는 멈출 수 있다.
- [x] 각 자동차에 이름을 부여할 수 있다. 전진하는 자동차를 출력할 때 자동차 이름을 같이 출력한다.
- [x] 자동차 이름은 쉼표(,)를 기준으로 구분하며, 이름은 5자 이하만 가능하다.
- [x] 사용자는 몇 번의 이동을 할 것인지를 입력할 수 있어야 한다.
- [x] 전진하는 조건은 기존의 자동차 이동 로직과 같다.
- [x] 경주 게임을 완료한 뒤 누가 우승했는지 알려준다. 우승자는 한 명 이상일 수 있다.
